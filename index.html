Einstellungen

✨ KI-Generierung – API-Key

Für die KI-Formulierungsgenerierung benötigst du einen Anthropic API-Key.
Kostet ca. 1–2 Cent pro Begehung (20 Abweichungen).
Key erstellen: console.anthropic.com → API Keys

Wird nur lokal auf diesem Gerät gespeichert, nie übertragen außer direkt an Anthropic.
📱 iPad → 💻 MacBook Workflow

iPad (vor Ort): Begehung erfassen, Fotos, Abweichungen
MacBook (Export): Word-Datei herunterladen

So überträgst du eine Begehung:
1. iPad: Begehung öffnen → 📤 Übertragen drücken → Code kopieren
2. MacBook: App öffnen → hier auf Importieren drücken → Code einfügen
3. MacBook: ⬇ Word Export drücken → fertig
